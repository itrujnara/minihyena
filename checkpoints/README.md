# MiniHyena Checkpoints
This directory contains pre-trained MiniHyena checkpoints.

## Bee Movie (`bee_movie`)
This tiny model knows the first sentence of the Bee Movie script by heart and can recall any part of it.
### Model setup
* `d_model`: 512
* `l_max`: 128
* `vocab_size`: 512
* `blocks`: `aah`
